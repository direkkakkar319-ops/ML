Interactive explainer: forward vs reverse mode automatic differentiation on a neural network node

Forward mode: sweep left→right, carrying ∂v/∂x₁ (tangent) forward at each step.
Example:  f(x₁, x₂) = ln(x₁) + x₁·x₂ − sin(x₂)  at  x₁=2, x₂=5  → find ∂f/∂x₁
Forward mode — when to use
Few inputs, many outputs (e.g. sensitivity analysis, Jacobian columns). Cost scales with inputs.
Reverse mode — when to use
Many inputs, few outputs (e.g. neural net with millions of params, one loss). Cost scales with outputs.
